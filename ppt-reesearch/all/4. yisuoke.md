## z.ai

- 预览：https://chat.z.ai/space/w0db071rbuw1-ppt
    
- 结果：
    
    - 输出的 slides 效果不错，虽然艺术性不行，但是中规中矩 Web 风。
        
    - 每一页的元素都可以进行手动修改或者 AI 修改，也可以整个页面进行整体调整
        
    - AI Slides 模式，3 个 tools 默认开启：Web Search，Image Search，Slides Maker。很实用。
        
    - 每次整体修改会记录版本
        
    - 迭代 4 个版本后，超对话上下文长度，需要重开新对话

## MinMax Agent

- 结论：输出效果不错，PPT 有动效，就是没有看到局部修改、编辑的功能，ppt 生成后会为其部署一个 http url 进行访问，回去自己查询相关图片补充到 ppt 中，并且最后会把所有格式（ppt,pdf,md...)都打包成一个 zip 包。
    
- 预览：https://3lnnwombn2bh.space.minimax.io/
