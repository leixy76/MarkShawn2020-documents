
### 1. 智能体 Zulu

###### （ 1 ）功能简介

只需自然语言描述，Zulu 将模拟人类工程师通过自主决策、工具使用等能力进行工程内的多文
件代码修改，完成编码任务，并与工程师充分协作和同步， **帮助工程师实现从思维到代码的无
缝转换** 。工程师的工作重点将从“写代码”转移到“决策”和“集成”。

**Zulu 适合的开发场景：**

```
1.端到端实现需求
2.参考已有逻辑实现新业务
3.实现跨文件/跨区块规律性编码
4.精准代码的定位，并分析
5.模糊问题排查解决
6.错误日志定位与修复
```
**Zulu 亮点能力：**

1. **深度理解代码库**
    Zulu 会自动调用代码搜索、文件读取等工具，理解您的代码库。这使得 Zulu 具备了理解您
    的业务逻辑的能力，您只需说出业务需求，Zulu 通过代码库搜索、阅读操作，找到对应代
    码，就可以理解您的意图，分析代码现状，为后续实现您的需求做好准备。
2. **自动执行命令**
    Zulu 会自动执行终端命令，进行开发环境设置、程序运行、测试、故障排查等工作。运行
    命令前，您需要手动点击执行。通过命令执行工具，Zulu 具备了完全使用您的开发环境的
    能力，您可以尝试将需要自己手动执行的命令交给 Zulu 来完成。
3. **代码变更精准采纳**
    Zulu 支持需求级多文件修改，生成后会在编码区为你通过 Diff 形式展示变更，同时支持代
    码片段级与文件级采纳，也可以通过会话维度操作回退。
