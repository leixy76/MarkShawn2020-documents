**「我真绷不住了」**

  

我觉得马卡龙（Macaron）这个产品太牛逼了。

我本来以为MyShell就已经是AI产品脱实向虚的极致了，但没想到还有高手🥵

在马卡龙面前，孙宇晨和Web3都算实体经济了。

这玩意儿最近热烈炒作，大规模投放，号称「全球首个Personal Agent」。

我真下载了，用了几分钟，给我整笑了。

马卡龙里边的Agent，大部分是AI生成的文字网页。连小游戏都不是，真是文字网页。

这事情太幽默了。我从来没见过一个产品，编故事都这么不用心。

打开马卡龙，首先一个聊天界面。左滑，就是Agent Store界面。里面有很多官方推荐的Agent，我点进去了「反诈助手」（名字真叫Anti-Fraud Guide）。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/iahp9f3MAqMR8gibHPU9YTMic3TvNj76AzBtKPCBxpHWF6rAwXynlwFlMdj7kj12xOFicpRwzr2jx5d3QFcssfU1Ww/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1)

它就是一个纯文字网页。真的是一个网页，AI 生成的，不带骗你的。朋友们，你懂我的感受吗？

我又点进去一个「绿色浪潮能源」（名字叫GreenWave Energy），我以为是个啥高级货。

它是一个看着像NGO官网的网页，里面甚至还有团队成员介绍，CEO、CTO 一应俱全。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/iahp9f3MAqMR8gibHPU9YTMic3TvNj76AzBXLOof8IWyFUcW9LZa6yIY2wbLkfGfUaZSCWcHricK0sc6ZwbgbqIoxA/640?wx_fmt=png&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1)

但这些全是编的。我上网搜了，真有GreenWave这个组织，但人家和马卡龙生成的这玩意毫无关系。

用到这里，我心里的感想只剩下佩服了。做出来这么牛逼的产品，还能高强度宣发，跟各路老师谈笑风生，哥们心理素质太强大了。

我抱着最后一点期望，回到聊天界面，叫马卡龙提醒我「明天下午六点钟要去海淀参加活动。」

马卡龙直接创造了一个「行程小助手」Agent。然后，我还是需要手动在这个工具里再设置一遍时间。设置完了，从昨天设置提醒到现在，马卡龙没有任何提醒。

并不意外，只能说是。

我真的很少被一个产品逗乐过。用上马卡龙，我确实绷不住。

你穿透它幽默的产品表象，触及灵魂，会发现bro讲了一个合成数据感非常强的故事。

我称之为「合成叙事」。

一级叙事，是 Vibe Coding。这是一个非常确定的趋势，像 Cursor 、Claude Code这种产品，是真的能干活。

二级叙事，是 Vibe Coding 社区。 这就是一个二级叙事了，一个「信则有，不信则无」的故事。「抽象需求、定义产品」是一种极度稀缺的能力。这玩意儿不像拍短视频，所见即所得，而是需要抽象思考，门槛很高。

普通人用AI写代码也做不出来好产品。我觉得Vibe Coding 社区的天花板是半成品4399小游戏。

但二级叙事毕竟还是有可能的，说不定大伙不爱刷抖音，爱上YouWare做生日贺卡了捏？

YouWare也就图一乐，真讲故事还得看马卡龙。

马卡龙整了一个三级叙事。它在 Vibe Coding 社区这个故事之上，又叠了一层，想要把社区里的作品，变成一个 Agent Store。

这已经是拿合成数据合成数据了，纯纯的人体蜈蚣。一个建立在Vibe Coding二次方故事之上的故事。

讲故事讲到这个份上，马卡龙已经超越Web3了。

我之前觉得 [MyShell是在搞脱实向虚](https://mp.weixin.qq.com/s?__biz=Mzk4ODYxNDE2OQ==&mid=2247483975&idx=1&sn=916ba72f700bb572da69eec7f0726b0e&scene=21#wechat_redirect)。有熟悉币圈的老哥纠正我说， MyShell在币圈绝对算实体经济。

确实。 MyShell好歹是一个逻辑闭环的产品。开发团队真发了币，你在上面做的 Agent，可以发布出来，有用户使用，创作者能挣Shell币，真能换成USDT。它的故事是闭环的。

马卡龙呢？它也讲创作者经济，但它只能送积分。用户用你的Agent，你拿积分，拿了积分可以去创造更多的Agent。

这打法纯机械学习Web3。问题是，你一个Web2产品，流动性是从哪来的？从创始人到处炒作导流吗？没有稳定的流动性啊朋友们。

一个AI产品，做得比Web3项目还虚，马卡龙也算作出创新了。

我开始怀念孙宇晨了。

孙割当年骗到投资后，搞过一个叫「陪我」的APP。这产品非常简单粗暴：陌生人社交语音房，男的付费找女的陪聊。

虽然糙，但你无法否认这产品真有需求，后来陪我APP也确实因为涉黄被下架了。

和马卡龙比起来，陪我APP也绝对算是实体经济。

最幽默的是，一个虚假成这个样子的产品，你只要下载用五分钟，就知道它是个编故事都不用心的东西。

但那么多媒体老师，居然真的能够一本正经地分析它有多么牛逼，它的价值在哪里。

还能创始人聊起来，一个敢问，一个敢答，从缸中之脑一路聊到强化学习。

所以有时候啊，媒体老师不要怪网友黑媒体。有时候纯属自找的，新闻学魅力时刻。

这里点名批评36氪的采访[扎克伯格想做的Agent，这个中国年轻人先做出来了](https://mp.weixin.qq.com/s?__biz=MzkyNDEzOTAzNg==&mid=2247486460&idx=1&sn=15983801ca6cfd371aa3b6ebc9adfa6b&scene=21&click_id=2#wechat_redirect)。

还他妈扎克伯格呢？

扎克伯格看到马卡龙都乐了。小扎心想，小伙子，我搞不成的元宇宙，未来是你的了。

36小编甚至不知道Claude Code怎么拼写的。就这水平，加入每日人物得了，还写啥AI呢。
